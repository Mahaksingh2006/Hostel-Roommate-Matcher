# Hostel Roommate Preference Matcher

## Description

A full-stack web application that helps hostel students find compatible roommates based on their preferences.

## Tech Stack

- Frontend: React + Vite
- Backend: FastAPI
- Database: MySQL
- ORM: SQLAlchemy
- Styling: Tailwind CSS
- Version Control: Git & GitHub
